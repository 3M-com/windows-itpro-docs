HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer!NoTaskGrouping

|  | Path |
|--|--|
| **CSP** | Not available. |
| **GPO** | **User Configuration** > **Administrative Templates** > **Start Menu and Taskbar** |
